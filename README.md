# Simple RISC-V Simulator (I & R formats only)

A beginner-friendly RISC-V assembly simulator for Computer Architecture course.

### Features
- Web-based editor (like simplified Venus)
- Supports R-type (`add`, `sub`, `and`, `or`, `xor`) and I-type (`addi`, `beq`, `bne`) instructions
- Step-by-step execution with visual register and memory view
- Good error messages with line numbers
- Clean and easy-to-understand code (student level)

### How to Run
1. Clone or download the repository
2. Open `index.html` in any browser
3. Write RISC-V code and click **Run All** or **Step**

### Technologies
- HTML + CSS + JavaScript (No external libraries)



---

**Note**: This is a student-level implementation. Easy to read, modify, and extend.
